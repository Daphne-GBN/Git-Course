# 01 - Getting Started

Before you write a single line of tracked code, Git needs to be installed and
you need to tell it who you are. This section covers:

- `git --version`
- `git config --global user.name` / `user.email`
- `git config --global --list`

These are one-time (or one-time-per-machine) setup steps, not things you
repeat per project.
